processes

pipeline

capture → video edit → photo edit → compose → output

capture

record source material; allow sensor artifacts when they add meaning.

inputs: phone video inputs: field audio inputs: found footage

tools: iPhone 13 Pro, Zoom H1n, ND filter

  • 4K/24; lock exposure; disable HDR; minimal stabilization.
  • ISO ≤ 800; above → lean into grain (Sensor Noise).
pitfalls
  • Over-stabilization causes wobble. Avoid auto HDR black crush.

example: sensor noise · glass city ↗

video edit

trim, stabilize lightly, curate artifacts; maintain rhythm over polish.

tools: Premiere Pro / Resolve

  • Sequence 24 fps; color temp fixed; avoid heavy NR.
  • Export master ProRes 422; delivery H.264 (see recipes).
photo edit

extract stills; emphasize texture; prep for print.

tools: Photoshop

  • Curves S-shape (3–5%); selective clarity; keep filmic blacks.
  • Resize to print DPI; soft proof target paper.
compose

collage & sequencing; micro-rhythm decides order; accept asymmetry.

  • Board vs timeline: choose by story density.
output

print & video delivery standards.

  • Print: Hahnemühle Photo Rag 308; perceptual intent; embed profile.
  • Video: 1080×1350 / 1920×1080; H.264 2-pass 12–16 Mbps; -1 dBTP, -14 LUFS.
recipes

SN–Still Extract: PNG16 → PS curve/clarity → soft proof → TIFF w/ profile.

IG 1080×1350 Export: H.264 2-pass 12–16 Mbps; audio -14 LUFS; -1 dBTP.

changelog
  • 2025-11-10 — added SN still-extract curve; updated IG export bitrates.
glossary

artifact curation — treating sensor errors as material to shape.

soft proof — simulate paper/profile before printing.
